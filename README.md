## challenge sanar

Create a terraform.tfvars with your credentials and default region.

**Example**:

* AWS_ACCESS_KEY = ""
* AWS_SECRET_KEY = ""
* AWS_REGION = ""

## Getting Started

 * terraform get -update
 * terraform init
 * terraform plan
 * terraform apply

 ### Dependencies

* Terraform.
* tfenv
* Account AWS
* AWS S3 Bucket backend

### Executing program

* terraform apply < file.tf >
